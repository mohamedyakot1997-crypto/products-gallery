# Products Gallery 🛍️

موقع جاليري منتجات احترافي للعرض والإدارة

## المميزات ✨

- ✅ عرض جاليري المنتجات بشكل جميل
- ✅ تفاصيل كل منتج (الصور، الوصف، السعر)
- ✅ لوحة تحكم كاملة لإدارة المنتجات
- ✅ إضافة/تعديل/حذف المنتجات
- ✅ رفع صور متعددة لكل منتج
- ✅ تعديل الأسعار والتفاصيل
- ✅ زرار WhatsApp لكل منتج
- ✅ لوحة تحكم محمية بكلمة مرور
- ✅ واجهة سهلة وجميلة

## البنية 📁

```
products-gallery/
├── index.html           (الصفحة الرئيسية للعملاء)
├── admin.html           (صفحة لوحة التحكم)
├── css/
│   └── style.css        (التنسيقات)
├── js/
│   ├── app.js          (كود العرض الرئيسي)
│   └── admin.js        (كود لوحة التحكم)
├── data/
│   └── products.json   (بيانات المنتجات)
└── README.md
```

## كيفية الاستخدام 🚀

### للعملاء:
1. افتح صفحة الجاليري الرئيسية
2. اضغط على أي منتج لشوف التفاصيل
3. اضغط على زرار WhatsApp الأخضر للتواصل مباشرة
4. سيتم فتح WhatsApp مع رسالة مخصصة بالمنتج

### لإدارة المنتجات:
1. افتح لوحة التحكم
2. أدخل كلمة المرور (أرقام من 1-5)
3. أضف منتجات جديدة
4. عدّل الأسعار والصور
5. احذف المنتجات القدي��ة
6. احفظ التغييرات

## التقنيات المستخدمة 💻

- HTML5
- CSS3 (Responsive Design)
- JavaScript (Vanilla JS)
- JSON (لتخزين البيانات)
- WhatsApp API
